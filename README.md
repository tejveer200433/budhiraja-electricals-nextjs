# Budhiraja Electricals

Single-page company website migrated to Next.js 15, React 19, Tailwind CSS 4 and TypeScript.

## Run locally

1. Install dependencies with `npm install`.
2. Copy `.env.example` to `.env.local` and add the Supabase and SMTP values used by the contact form.
3. Restart the dev server after creating or updating `.env.local`.
4. Start development with `npm run dev`.
4. Open `http://localhost:3000`.

Use `npm run build` to create a production build.
